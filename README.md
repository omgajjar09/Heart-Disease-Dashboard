# ❤️ Heart Disease Dashboard

## 📌 Project Overview

The **Heart Disease Dashboard** is an interactive Power BI report designed to analyze clinical patient data and identify factors related to heart disease outcomes. The dashboard provides meaningful insights into patient demographics, medical conditions, laboratory measurements, and survival status using interactive visualizations.

This dashboard helps users understand patient health patterns and supports data-driven healthcare analysis.

---

# 📊 Dashboard Features

- Interactive KPI Cards
- Patient Demographic Analysis
- Medical Condition Analysis
- Clinical Measurement Analysis
- Survival (Death Event) Analysis
- Dynamic Filters (Slicers)
- Interactive Charts and Visuals

---

# 📁 Dataset Information

**Dataset Name:** Heart_Disease_Dataset.xlsx

**Total Records:** 299

### Dataset Columns

| Column Name | Description |
|-------------|-------------|
| count | Patient Record ID |
| age | Age of Patient |
| Hypertensions | Hypertension Status |
| muscle_damage_scale | Muscle Damage Scale |
| diabetes | Diabetes Status |
| ejection_fraction | Heart Pumping Efficiency (%) |
| high_blood_pressure | High Blood Pressure Status |
| platelets | Platelet Count |
| kidney_marker | Kidney Function Marker |
| serum_sodium | Sodium Level in Blood |
| sex | Patient Gender |
| cancer | Cancer Status |
| time | Follow-up Time |
| DEATH_EVENT | Patient Survival Status |

# 📈 Dashboard Analysis

## Patient Demographics

This section provides an overview of patient characteristics such as:

- Age Distribution
- Gender Distribution
- Total Patients

These visuals help identify the demographic profile of the patients included in the dataset.

---

## Medical Condition Analysis

The dashboard analyzes major medical conditions affecting patients, including:

- Hypertension
- High Blood Pressure
- Diabetes
- Cancer

This analysis helps compare the occurrence of different health conditions among patients.

---

## Clinical Measurements

Several important clinical parameters are analyzed, including:

- Muscle Damage Scale
- Ejection Fraction
- Platelet Count
- Kidney Marker
- Serum Sodium Level

These measurements provide insights into patient health and heart function.

---

## Survival Analysis

The dashboard includes survival analysis using the **DEATH_EVENT** field.

It compares:

- Survived Patients
- Death Cases

This helps understand the distribution of patient outcomes in the dataset.
